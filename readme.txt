Select images just with a single click to show them in colorbox popup from admin ajax grid.

== Description ==
Selected Colorbox Image plugin allow you to select one or more or all images to show it/theme in Colorbox popup. Selected Colorbox  Image provide you datatable based image list with title, image and ID list, just select images you want to display in Colorbox popup and that\'s all.

== Installation ==
Follow these steps to install Selected Colorbox Image plugins to your wordpress site:
1. Upload allimg.zip to your site/wp-content/plugins/ directory
2. Activate Selected Colorbox Image plugins from your admin dashboard
3. Once you activate the plugin, you will find \'Colorbox\' menu in your left menu panel
4. Go there and select whatever images you want to show in colorbox popup
